# DATA PATH v3.1 正式版

90日間のデータアナリスト学習を管理する静的Webアプリです。

## ファイル構成

```text
index.html
styles.css
app.js
glossary-data.js
glossary-index.js
glossary-ui.js
README.md
CHANGELOG.md
```

## v3.1の主な機能

- 用語カテゴリは初期状態ですべて折りたたみ
- 5語を超えるカテゴリはプレビュー表示
- 関連用語リンクと未登録関連用語の検出
- 用語追加後3日間のNEW表示
- お気に入り
- 初登場Day、理解度、登録日、更新日
- カテゴリ別理解度バー
- 理解度1〜2の復習候補
- ホーム画面へ最近追加、理解度、復習候補を表示
- DAY終了メッセージのワンクリックコピー
- JSONバックアップ／復元

## GitHubへの反映

ZIPを展開し、リポジトリ直下へ8ファイルをまとめてアップロードしてください。
旧 `glossary.js` は削除し、新しい3ファイルへ置き換えます。

## 注意

更新前にDATA PATHからJSONをエクスポートしてください。既存データは同じlocalStorageキーを使うため引き継がれますが、バックアップを推奨します。
