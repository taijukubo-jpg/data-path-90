# DATA PATH v2.0

90日間のデータアナリスト学習を管理する、GitHub Pages向けの静的Webアプリです。

## ファイル構成

```text
data-path-90/
├─ index.html   # 画面構造
├─ styles.css   # デザイン
├─ app.js       # 学習計画・進捗管理・ログ・AI連携
└─ README.md    # 説明書
```

## 主な機能

- 日付を基準にした今日のDay表示
- 学習不可日の除外
- Dayごとのタスク管理
- 前倒し／遅れの判定
- 3行レポート
- 学習ログ一覧
- 専門用語集（検索・カテゴリ・編集・削除）
- 生成AI回答からの用語一括取り込み
- 生成AI向け引き継ぎ文の作成とコピー
- JSONバックアップ／復元

## GitHubへの反映

### 初回

リポジトリ直下へ、次の4ファイルをアップロードします。

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

### 既存ファイルとの関係

- `index.html`：既存ファイルを差し替え
- `README.md`：既存ファイルを差し替え
- `styles.css`：新規追加
- `app.js`：新規追加

最終的なリポジトリ直下は次の状態です。

```text
README.md
app.js
index.html
styles.css
```

## 生成AI連携のテスト

1. DATA PATHを開く
2. 「引き継ぎ文を作成してコピー」を押す
3. メモ帳などへ貼り付ける
4. 次の文字から始まれば成功

```text
【DATA PATH｜学習状況の引き継ぎ】
```

## データ保存

進捗データはブラウザの `localStorage` に保存されます。

ブラウザの初期化やPC変更に備えて、週1回を目安にJSONをエクスポートしてください。


## 用語集への追加方法

### 通常追加

1. 「＋ 用語を追加」を押す
2. 用語・カテゴリ・意味を入力
3. 保存する

### 生成AIの回答から追加

生成AIへ、次の形式で回答するよう依頼します。

```text
用語: DAX
カテゴリ: Power BI
意味: Power BIで集計や分析を行うための数式言語
実務例: 応答率、AHT、ASAなどのKPIを計算する
関連用語: メジャー, 計算列, フィルターコンテキスト
```

DATA PATHの「生成AIの回答から手早く追加する」へ貼り付けて取り込みます。

複数用語を追加するときは、用語同士を空行または `---` で区切ります。
