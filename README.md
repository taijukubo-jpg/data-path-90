# DATA PATH v3.1.1 正式版

90日間のデータアナリスト学習を管理する静的Webアプリです。

## ファイル構成

```text
index.html
styles.css
app.js
glossary-data.js
glossary-index.js
glossary-ui.js
README.md
CHANGELOG.md
```

## v3.1の主な機能

- 用語カテゴリは初期状態ですべて折りたたみ
- 5語を超えるカテゴリはプレビュー表示
- 関連用語リンクと未登録関連用語の検出
- 用語追加後3日間のNEW表示
- お気に入り
- 初登場Day、理解度、登録日、更新日
- カテゴリ別理解度バー
- 理解度1〜2の復習候補
- ホーム画面へ最近追加、理解度、復習候補を表示
- DAY終了メッセージのワンクリックコピー
- JSONバックアップ／復元

## GitHubへの反映

ZIPを展開し、リポジトリ直下へ8ファイルをまとめてアップロードしてください。
旧 `glossary.js` は削除し、新しい3ファイルへ置き換えます。

## 注意

更新前にDATA PATHからJSONをエクスポートしてください。既存データは同じlocalStorageキーを使うため引き継がれますが、バックアップを推奨します。


## v3.1.1 データ移行について

- 既存のブラウザ内用語データをそのまま保持します。
- 初回起動時、旧用語データを `data-path-glossary-backup-before-3.1.1` に退避します。
- 既存用語と初期カタログを用語名で照合し、未登録分だけ追加します。
- 同名用語は重複登録しません。
- v3.1で用語が0語に見えた原因だった `glossary-data.js` の構文エラーを修正しています。

## 生成AI連携

「学習状況を引き継ぐ」と「今日の学習終了を報告する」を同じカードへまとめています。
どちらも、文章作成 → プレビュー表示 → コピー → ChatGPTを開く、という同じ流れで操作できます。
