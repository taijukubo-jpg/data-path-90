# DATA PATH v3.0

90日データアナリスト学習を管理するGitHub Pages向けWebアプリです。

## ファイル構成

```text
index.html      画面構造
styles.css      デザイン
app.js          学習計画・進捗・ログ・AI引き継ぎ
glossary.js     用語集・関連リンク・用語カタログ
README.md       説明書
```

## v3.0の主な変更

- 用語集の処理を `glossary.js` へ分離
- カテゴリ単位の折りたたみ表示
- 5語以上のカテゴリは先頭5語だけ表示
- 関連用語をクリックして該当用語へ移動
- 未登録の関連用語を警告表示
- 過去に提案した用語と関連用語を初回起動時に一括登録
- 最近追加した用語を表示
- ホーム画面に登録用語数を表示
- 既存のlocalStorageとJSONバックアップを維持

## GitHubへの反映

リポジトリ直下へ次の5ファイルをアップロードします。

- `index.html`
- `styles.css`
- `app.js`
- `glossary.js`
- `README.md`

既存の同名ファイルは削除せず、そのままアップロードして上書きします。`glossary.js`だけは新規追加です。

## 初回確認

1. GitHub Pagesを開く
2. 用語集までスクロールする
3. カテゴリが折りたたまれていることを確認する
4. 関連用語をクリックし、該当用語へ移動することを確認する
5. 未登録の関連用語が0語になっていることを確認する
6. JSONをエクスポートしてバックアップする

## データ保存

進捗・3行レポート・用語集はブラウザの `localStorage` に保存されます。既存のデータは引き継がれ、カタログに存在する未登録用語だけが追加されます。
